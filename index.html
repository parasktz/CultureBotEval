⚙ Σύνδεση Firebase
Ακολούθησε τα βήματα, βάλε τους κανόνες ασφαλείας, και επικόλλησε τα κλειδιά.
1
Πήγαινε στο console.firebase.google.com → Add project
2
Build → Realtime Database → Create database → επίλεξε region → Start in test mode → Enable
3
Realtime Database → Rules → διέγραψε τα πάντα → επικόλλησε αυτούς τους κανόνες → Publish:
{ "rules": { "sessions": { ".read": true, "$sessionId": { ".write": "!data.exists()", "evaluations": { ".write": true } } } } }
✓ Αυτοί οι κανόνες εμποδίζουν τη διαγραφή δεδομένων από χρήστες.
4
⚙ Project Settings → General → Your apps → Add app → Web → Register → αντέγραψε το firebaseConfig
5
Επικόλλησε τις τιμές παρακάτω και πάτα Αποθήκευση
apiKey
projectId
databaseURL (από Realtime Database)
authDomain
appId
